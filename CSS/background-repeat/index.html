round模式下,div容器包含背景图片的行数和其高度有关

div高度在(0~<3/2)*图片高度 的范围时,为1行,不足一个图片实际高度的话图片会被压缩,高出一个图片高度会被拉伸

div高度在(3/2~<5/2)*图片高度 的范围时,为2行

div高度在(5/2~<7/2)*图片高度 的范围时,为3行

以此类推,宽度同理

space模式下,div容器包含背景图片的行数和其高度有关

div高度在(0~<2)*图片高度 的范围时,为一行,不足一个图片实际高度的话图片会被裁剪,高出一个图片实际高度会留空

div高度在(2~<3)*图片高度 的范围时,为2行,大于两个图片实际高度的话图片中间会留空白,图片紧贴上下div上下边

以此类推,宽度同理
